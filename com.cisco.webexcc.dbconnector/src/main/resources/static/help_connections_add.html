Add Database Connection - Help Guide

Overview

Use this form to register a new external database connection. These connections are used by SQL endpoints to execute queries.

Fields

  • Connection Name: A unique, descriptive name for this connection (e.g., Sales DB Production).
  • Environment:
    • DEV: For development and testing purposes.
    • UAT: For user acceptance testing.
    • PROD: For production systems.
  • Database Type: Select the vendor of your database (MySQL, SQL Server, Oracle, PostgreSQL).
  • JDBC URL: The standard JDBC connection string.
    Examples:
    • MySQL: jdbc:mysql://hostname:3306/dbname
    • SQL Server: jdbc:sqlserver://hostname:1433;databaseName=dbname
    • Oracle: jdbc:oracle:thin:@hostname:1521:sid
    • PostgreSQL: jdbc:postgresql://hostname:5432/dbname
  • Username & Password: Credentials with sufficient privileges to execute the queries you plan to define.

Testing

Click the Test Connection button to verify that the application can connect to the database with the provided details. The output console will show the result of the connection attempt.
