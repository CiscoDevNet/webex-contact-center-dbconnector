Dashboard Help

Welcome to the Webex Database Connector Dashboard. This page provides an overview of your connector's activity and health.

Environment Stats

The top section shows real-time statistics for your configured environments (e.g., DEV, UAT, PROD):

  • Hits: Total number of successful requests processed.
  • Failures: Number of failed requests.
  • Status: Color-coded indicators show the environment type (Blue for PROD, Green for UAT, Yellow for others).

Detailed Activity

The main section lists detailed activity for each environment:

  • Endpoint Activity: Shows which API endpoints are being accessed.
  • Hit Counts: Number of times each specific endpoint was called.
  • Last Execution: Timestamp of the most recent activity.
Note: You can use the Cleanup button to remove stats for endpoints that no longer exist, or Reset to clear all counters for an environment.
