SQL Management - Help Guide

Overview

The SQL Management page is the central hub for creating, viewing, and managing the SQL queries that power your Contact Center integrations. Each query is exposed as a secure REST API endpoint.

Managing SQL Statements

Adding a New Statement

Click the "Add SQL Statement" button to create a new query. You will need to provide:

  • Name: A unique identifier for the query (e.g., getCustomerById). This becomes part of the API URL.
  • Connection: The database connection to run the query against.
  • SQL: The actual SQL query. Use named parameters like :id for dynamic inputs.
Note: You can only add new statements to DEV environments directly. To move them to UAT or PROD, use the Deploy feature.

Actions

For each statement in the list, you can perform the following actions:

  • Test (Beaker Icon): Opens the Test Console pre-loaded with this query to verify it works.
  • Deploy (Rocket Icon): Promotes a DEV query to UAT or PROD environments.
  • Edit (Pencil Icon): Modify the SQL or description (DEV only).
  • Delete (Trash Icon): Remove the statement.

Filtering

Use the search bar at the top of the table to quickly find statements by name, environment, or SQL content.
